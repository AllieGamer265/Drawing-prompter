Drawing Prompter

Responde unas preguntas y te doy 3 ideas para dibujar (buscadas en la web).

1) Colores (elige hasta 4)

2) Materiales disponibles

3) Tiempo disponible

4) Dificultad

5) Estilo preferido

6) Tema preferido

7) Estado de ánimo / detalle adicional (opcional)

Ideas

Proyecto local — usa npm install y npm start para ejecutarlo.
